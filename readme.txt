Adds a header tag to your WordPress installation showing google who the author is of the post.

== Description ==

This adds a link to the header of your WordPress installation showing google who the author of the post is.

== Installation ==

1. Download the latest zip file and extract the `google-rel-author` directory.
2. Upload the this directory inside your `/wp-content/plugins/` directory.
3. Activate 'Google rel author Plugin' on the 'Plugins' menu in WordPress.
4. Add your Google+ number to your profile page.
5. ???
6. Profit.

== Frequently Asked Questions ==

= It doesn't show up? =

Make sure that you have added your google+ number to your profile page. This must be done for every author that has a Google+ profile.
